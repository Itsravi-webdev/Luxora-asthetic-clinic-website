# ✦ Luxora Aesthetics — Reveal Your Natural Glow ✦

**[🌍 View Live Website](https://luxora-aesthetics.netlify.app)**

Welcome to the official repository of **Luxora Aesthetics**, an ultra-premium, modern, and highly responsive single-page web experience designed for luxury cosmetic and aesthetic skin clinics. 

Where medical science meets artistic precision, this web experience reflects that harmony through a sophisticated dark-and-gold aesthetic, fluid animations, and interactive client engagement tools.

---

## 🎨 Visual Identity & Theme
Luxora Aesthetics is styled with a bespoke design system that communicates luxury, professionalism, and clinical excellence.
* **Deep Obsidian & Slate Backgrounds (`#0B0C10`, `#1F2833`)** for a sleek, immersive dark mode experience.
* **Warm Metallic Gold Gradients (`#D4AF37`, `#FFD700`, `#AA7C11`)** as accent colors, evoking luxury and prestige.
* **Premium Typography:**
  * Serif headers: **Cormorant Garamond** (timeless, elegant, and editorial)
  * Sans-serif body: **Outfit** (modern, clean, and clinical)
* **Glassmorphism UI elements** using advanced backdrop-filters to give a modern, layered visual structure.

---

## ✨ Key Features
* **Custom Logo Preloader:** A sleek pre-loading transition sequence showcasing the brand name with a loading progress bar to enhance perceived performance.
* **Cinematic Hero Banner:** Parallax background elements, glowing ambient floating orbs, and animated entrance states.
* **Interactive Statistics Counter:** Real-time counters showing years of experience, successful procedures, and client satisfaction numbers that animate when scrolled into view.
* **Infinite Text Marquee Strip:** An infinite scrolling slider showing signature clinical specialities.
* **Interactive Before & After Comparison Slider:** Allows potential clients to slide between "Before" and "After" states of treatment results dynamically.
* **Masonry Gallery with Lightbox:** A modern, asymmetrical image gallery showcasing clinic spaces and treatment phases, complete with a fullscreen interactive lightbox viewer.
* **WhatsApp Consultation Booking Integration:** A beautifully designed booking portal that validates appointments and generates pre-filled WhatsApp consultation requests directly to the clinic.
* **Fluid Scroll Animations:** Handcrafted CSS transitions triggered on scroll via Intersection Observer for a polished, cinematic feel.

---

## 🛠️ Technology Stack
* **Structure:** Semantic HTML5
* **Styling:** Custom Vanilla CSS3 (Variable-driven design system, Flexbox, Grid, custom keyframes)
* **Interactions:** Modern Vanilla JavaScript (ES6+, Intersection Observer API, event handling, dynamic DOM updates)
* **Iconography:** Hand-crafted SVG icons for crisp scaling and zero layout shifts.

## 🚀 Getting Started

### Prerequisites
To run this project locally, you don't need any complex build tools. A standard modern web browser (Chrome, Safari, Edge, or Firefox) is all that is required.

### Local Development
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Itsravi-webdev/Luxora-asthetic-clinic-website.git
   ```
2. **Navigate into the directory:**
   ```bash
   cd Luxora-asthetic-clinic-website
   ```
3. **Open the project:**
   * Simply double-click `index.html` to open it in your default browser.
   * Or use a local development server like **Live Server** in VS Code, or python's built-in server:
     ```bash
     python -m http.server 8000
     ```
     Then navigate to `http://localhost:8000`.
